# BaseFaq UI/UX Workspace

This folder is the delivery root for the BaseFaq Design System and UI-UX agent.

## Store Here

- Product flow maps
- Wireframes and experience specs
- Design-system notes
- Accessibility reviews
- UX research summaries
- UI copy decks

## Keep Out Of This Folder

- Runnable frontend code
- Backend contracts
- Infrastructure configuration

## Working Rule

All deliverables must stay in English and should reference the Demo6 Next.js TypeScript baseline when implementation guidance is needed.
